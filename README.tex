\section{Background}\label{background}

This Git repository offers a Quarto template for writing bachelor's,
diploma, and dissertation theses. Originally adapted from a LaTeX
template by \href{https://github.com/nmfs-opensci/quarto-thesis}{Eli
Holmes}, this version has been customized to meet the specific
requirements of Johannes Kepler University (JKU).
